js选择器总结

操作DOM树应该是我们做的最多的一些工作了，操作DOM树时，选择节点是必不可少的一项工作，下面我们就来说一说js里面常用的选择器（选择符）。
一般常用的
1，document.getElementById();参数接收一个id，返回一个唯一的节点。
2，document.getElementsByTagName()；参数接收一个标签名（可能描述不准确），返回一组节点（nodelist节点），以数组形式访问。
3，document.getElementsByClassName();参数接受一个类名，返回一组节点（nodelist节点），以数组形式访问。
4，document.getElementsByName()；参数接收节点的name属性值，返回一组节点（nodelist节点），以数组形式访问。

上面是三个基本的选择器，选择器可以组合使用。
