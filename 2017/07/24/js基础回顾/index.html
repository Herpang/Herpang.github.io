对基础的一些回顾

其实主要是js高程（红宝书）第三章的一些知识，在这里总结一下

数据类型

5种基本数据类型，Number，String，Boolean，Null，Undefined
1种复杂数据类型，Object
一些操作符
语句
函数

大概就是主要内容了，下面详细说

几个琐碎的点

1，js区分大小写
2，typeof操作符，返回值是一个字符串，可以识别 undefined，boolean，string，number，function，但是 null和object识别的不是很好， null的话会返回object，主要是区分不了null，undefined，array

Undefined和Null

这两个很容易混淆
Undefined和Null这两种类型都只有一个值，
undefined，在用var声明变量但并没有初始化赋值等操作时，这个变量的值就是undefined，没有声明的变量也会自动被赋予undefined值
null，表示一个空的对象指针，（这也是为什么用typeof操作符检测的时候会返回object）如果定义的变量准备在将来存储一个对象，那么一般将这个变量初始化为null，这时候我们只需要检查这个变量的值是否是null就可以判断这个对象是否已经保存值了，

Boolean

布尔类型有两个值，true和false（区分大小写）
一个转型函数Boolean（），可以将其它类型的值转换为布尔类型，下面是对应的转换关系
string 类型，非空的字符串—-true，空字符串—-false
number类型，任何非0数值—-true，0和NaN—-false
object类型，任何对象—-true，null—-false
undefined—-false

Number

1，很重要的一个类型，包括整数和浮点数值，支持8进制和16进制
2，浮点数值（包涵一个小数点的数值，需要的内存空间是整数的两倍）
3，一般较大的值，可以用科学计数法，1e3 = 1e+3 = 1000 1e-3 = 0.001
4，浮点数值的最高精度是17位小数，在计算的时候，精度不如整数，会产生舍入误差。
5，number最大大概是1.7e+308 最小5e-324，超过这个范围会显示infinity
6，Infinity-Infinity会返回NaN，判断是否是Infinity可以用isFinite（）函数，会返回布尔值
7，NaN，非数值，表示本来应该返回数值但未返回，为了防止抛出错误而出现，设计NaN的操作都会返回NaN
只有一个isNaN()方法可以判断是否是NaN
8，数值转换的方法 Number（）parseInt（）parseFloat（）
Number（）转换的过程比较复杂且不够合理，一般用的比较少
parseInt（）例如123rrr会解析成　　１２３，个人其实这个函数的主要作用还是将２进制，８进制，１６进制等格式的数据转换成10进制的，下面看两个例子

parseFloat（）只能解析10进制的值，不接受第二个参数，其他的和parseInt（）类似。

String

1，一般来说字符串是不可变的，如果要改变这个字符串的值，后台的过程是，先销毁原来的字符串，然后用另一个包涵新值的字符串填充这个变量
2，toString（） 这个方法会返回一个字符串的副本，除了null和undefined值之外所有的对象都有这个方法。
3，String（）这个方法我个人觉得相当于toString（）的加强版，可以识别null和undefined，执行过程—如果该值有toString（）方法，调用该方法—-如果是null返回null，如果是undefined返回undefined

Object

object对象其实是一组数据和功能的集合
1，object对象的实例都有以下几个基本方法和属性
constructor属性，指向创建该对象的函数（原型）
hasOwnProperty（propertyname）用于检查给定的属性是否是该对象自己的（而不是从原型上继承过来的）
isPrototypeOf()用于检查给定的属性是否是当前对象的原型
propertyIsEnumerable（）用于检查给定的属性能否被 for in枚举
toLocaleString（） 返回对象的字符串表示，并与对应的执行环境的地区对应
toString（）。。。
valueof（）。。。

一些操作符

1，主要讨论i++和++i的区别

主要区别就是 –i是先执行–i再执行相应的语句，而i–是先执行相应的语句再执行i–
2,位操作符，操作底层一点的2进制的，不多讨论
3，,/,%, 乘，除，取余，不多介绍了，大家都很熟悉。
4，*布尔操作符！，会返回一个布尔类型的值，看过一些大神的代码，会用！操作符来执行代码，在函数前面加个！，无论什么结果这个函数都会执行一边，略高端。
5，&&，||,都遵循短路操作
&&，如果第一个值为真，则返回第二个值，如果第一个值为假，返回第一个值
||，如果第一个值为真，则返回第一个值，如果第一个值为假，返回第二个值
6，语句不多说了，很基础，天天写，肯定不会忘，多看看for in
7，函数，一个点是arguments对象，在函数内部可以通过该对象来访问输入的参数

8,函数没有重载，同名的函数后面的会覆盖前面的
